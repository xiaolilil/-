目录
pages      所有的页面
components 所有公共的页面区域
layout     布局区域  header  footer  menu
js         项目的js
styles     项目的样式
api        项目的所有网络请求接口
utils      项目的工具函数
config     项目的依赖文件  layui  swiper  axios  jquery
assets     项目的所有静态资源 图片 图标
